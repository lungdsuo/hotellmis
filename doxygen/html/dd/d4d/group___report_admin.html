Hotel Holiday Report
A Simple Video Tutorial on How to Generate a Hotel Holiday Report


This report generates the hotel holidays report on the calendar year that was being setup by the hotel.
The report will display the holidays of the setup in the system which helps hotel take note while setting up promotional rates for holidays.

To generate a Hotel Holiday Report:
1.Select a specific date range then,
2.Click the Show Result button.
All the holidays for the specific year will display.
To download the file in spreadsheet like excel, just click the Download button.

Shift Report/Maintenace
A Simple Video Tutorial on How to Generate a Shift Report


This report prints out the transactions made by type for a specific user on shift. Administrators can print out reports for all users.
This helps the hotel keep track of the transactions created by users that is in the system.
This will show a transactions created by specific or all the users within a date range.
To generate a Shift Report/Maintenace:
1.Select a start then,
2.Select a date range then,
3.Select a user then,
4.Click the Go button.
The Shift Report/Maintenace for the specific date range will display.
To download the file in spreadsheet like excel, just click the Download button.
