시스템 설치 및 설정 안내

OTA 소프트웨어를 설치하기 위해서 다음의 구성이 지원됩니다.
WAMP

  • Windows
  • Apache
  • Mysql
  • PHP
or
LAMP
  • Linux
  • Apache
  • Mysql
  • PHP

Install and setup the your LAMP/WAMP server as per the apache/php websites http://www.apache.org and http://www.php.net

mysqladmin tool을 사용하여 hotelmis라는 데이터베이스를 생성합니다.
데이터베이스를 생성하고 hotelmis.sql을 사용하는 견본 공간을 만듭니다.
Hotelmis와 같은 데이터베이스 사용자를 생성합니다.



이제 configuration.inc.php를 편집합니다.
MySQL 서버에서데이터베이스, 사용자 그리고 비밀번호 를 설정하고 hotelmis.sql을 실행합니다. 그리고 데이터베이스 표와 처음 데이터를 생성합니다.

기본 계정은 “admin”이고 비밀번호는 “password”입니다.

configuration.inc.php 파일을 수정하고 설정합니다:

HOST 는 mysql 데이터베이스가 설치되는 IP 혹은 주인의 이름입니다.
define("HOST", "localhost");

PORT 는 소켓 번호이고, 기본값은 3306입니다.
define("PORT", 3306);

USER 는 mysql 데이터베이스에 접속하는 로그인 아이디입니다.
define("USER", "hotelmis");

PASS 는 사용자에 대한 비밀번호입니다.
define("PASS", "hotelmis");

DB 는 데이터베이스의 이름입니다.
define("DB", "hotelmis");

DEBUG 는 디버그 프린팅 정보와 관련한 것입니다.
define("DEBUG", 0);

TAXPCT 는 세금 계산에 대한 기본 비율입니다.
define("TAXPCT", 7);

SVCPCT 는 봉사료에 대한 서비스 비율입니다.
define("SVCPCT", 10);
