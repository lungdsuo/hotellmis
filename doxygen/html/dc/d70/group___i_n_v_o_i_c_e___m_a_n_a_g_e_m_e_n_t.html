  $qty,

&  $std_tax,
&  $std_svc,
&  $amt,
&  $tax,
&  $svc 
)

Check the rates for the selected item and return the billable amount for the product code. If the itemid is set, then std_amt, std_tax and std_svc must be set will be compared against the rateid. If the rateid does not contain the itemid then the amt will be set to std_amt, tax for std_tax, svc
roomid and itemid should not be used at the same item, roomid will update itemid.
Only one day charge is calculated at a time for a room and 1 room at a time. Multiple rooms require multiple registrations. Gross amount charge will return actual charge amount, if 3 breakfasts are free, then charge for 1 will be returned. If 3 are discounted, then charge amount for 3 x discount and 1 by normal rate for the day will be returned.

Parameters
$itemid[in/out] item id for - auto set for rooms.
$qty[in] quantity of itemid (not room)
$roomid[in] room id
$chg_date[in] The charge date "dd/mm/yyyy" format
$rateid[in/out] requested rate id requested
$std_amt[in/out] Standard amount charge, if not populated will be collected from the rate id
$std_tax[in/out] Standard tax amount, if not populated will be collected from the rate id
$std_svc[in/out] Standard service charge, if not populated will collected from the rate id
$amt[in/out] Update from the rate and charge
$tax[in/out] Update from the rate and std tax
$svc[in/out] Update from the rate and std service charge
Returns
gross amount charged
