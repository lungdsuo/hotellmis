系统安装和设置说明

需要以下配置来安装OTA酒店软件
WAMP

  • Windows
  • Apache
  • Mysql
  • PHP
或者
LAMP
  • Linux
  • Apache
  • Mysql
  • PHP

按照安装和设置了Apache/ PHP网站http://www.apache.org和http://www.php.net了您的LAMP/WAMP服务器

使用mysqladmin工具创建一个名为hotelmis的数据库
创建数据库和使用hotelmis.sql取样客房数据
创建数据库用户，如hotelmis/ hotelmis



现在编辑configuration.inc.php
设置定义为DB, USER 和 PASSWORD 在MySQL服务器的主机上，运行hotelmis.sql和创建数据库表和初始数据。

默认帐户是“admin”密码“password”

修改configuration.inc.php文件和设置定义

HOST为IP地址或者mysql数据库安装所在的主机名
define("HOST", "localhost");
PORT是插口编号，默认为3306
define("PORT", 3306);

USER 是登入mysql数据库的用户名
define("USER", "hotelmis");

PASS是用户的密码
define("PASS", "hotelmis");

DB是数据库名字
define("DB", "hotelmis");

DEBUG打开一些纠错打印信息
define("DEBUG", 0);

TAXPCT是默认的税率百分比
define("TAXPCT", 7);

SVCPCT是服务费的百分比
define("SVCPCT", 10);
