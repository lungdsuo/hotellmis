Every reservation and booking requires a booking rate be associated with the transaction.
If the booking rate is not specified, then the default rate for the room will be applied.
Default rates only apply to room items, all other items must have charge rate appled at time of adding to invoice
The rate administration is divided into 2 sections, rate information and rate item pricing.
When adding a new rate, only the rate information is available.
Only after the rate information is added and saved can the rate item pricing be added.

Press the Add Rate button to save the rate information to enable addition of rate item prices.

Each rate has rate items associated for pricing and packaging. For a promotional, customer or agent rate every product code or item will be available for selection.
For default rates, only room product code or item can be selected.
Items cannot be editted, only deleted and re-added to the rate.

Rate List
Select the Rates List button to go to the list of all the rates.
Click the magnifying glass icon beside a specific rate to view/edit that rate.


Standard Rates
Every room has a standard rate charge, the standard rate is for the room only. The pull down will only allow the selection of product codes or items if the rate type is Promotion, Agent, Customer and Fee.
