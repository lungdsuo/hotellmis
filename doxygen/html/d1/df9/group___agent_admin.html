The view/update screen for travel agencies.
Select the Agents List button on the right to view the agent list.

In order to facilitate transactions created by Travel Agencies, and also to process commission rebate to travel agencies, each agency must be known to the OTA Hotel software.
Select the magnifying glass icon beside the agent IATA to view/edit the travel agent.
Use the search criteria in the right panel to find agents, by name or by IATA number.
