get_hotelgallery ( &  $gallery,
  $pg = 0,
  $img = 0 
)

Get the hotel galleries

Parameters
$gallery[in/out] Output array
$pg[in] 0 for gallery page 1 for promo page, -1 for all
$img[in] 0 for image 1 for video, if pg = -1, this arg ignored
Note
output array is indexed and each element has the following members idx is from 0 to number or gallery images
$gallery[<idx>]['PicID'] - Pic ID
$gallery[<$idx>]['Title'] - Title of pic
$gallery[<$idx>]['Description'] - Description of pic
$gallery[<$idx>]['URL'] - URL of pic
Returns
number of pictures in gallery
