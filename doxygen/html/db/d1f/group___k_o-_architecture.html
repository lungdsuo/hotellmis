Introduction to OTA Hotel Architecture

시스템 구성

OTA 호텔 프로그램은 단순한 웹 기반의 호텔 인벤토리 관리 시스템입니다.
소프트웨어는 오픈소스로써 사적이거나 상업적 용도를 위해 무료로 제공됩니다.

이 OTA 호텔 프로그램은 일반 공중 라이센스에 의해 보호받습니다.

OTA 호텔 프로그램은 본래의 오픈소스 프로그램인 HotelMIS로부터 재개발되었습니다. 그러나 본래 소스코드는 거의 남지 않았습니다.
OTA 호텔 프로그램은 Open Travel Alliance 메세지 규격을 사용하는 e-Novate Pte Ltd e-Bridge 소프트웨어를 위한 시험용 플랫폼으로써 만들어졌습니다.
일반적으로 OTA 호텔 프로그램은
이 도서관은 소프트웨어의 실수요자가 예약에 대한 온라인 인터페이스를 제공하고자 e-Novate Pte Ltd e-Bridge의 서비스를 사용하는 어느 호텔, 소프트웨어 개발 회사이든지 별다른 허가 없이 사용될 수 있습니다.

The e-Novate Pte Ltd e-Bridge 소프트웨어 솔루션은 기업간 전자상거래나 기업과 소비자간 전자상거래를 위한 포괄적인 상호작용을 가능케하는 저렴한 연결이자 현재의 솔루션입니다.
e-Bridge 제품에 대한 더욱 자세한 정보를 위해서는 e-Novate의 sales@e-novate.asia에 문의하십시오.

OTA 호텔 사용자를 위한 무료 온라인 호텔 예약 솔루션

OTA 호텔 소프트웨어 사용자는 설립 비용이 필요 없는, 그리고 자유로운 접근이 가능한 OTA 호텔 소프트웨어에 직접 연결되는 온라인 예약 사이트에 적합합니다.
이것은 가격을 낮춰 모든 등급의 호텔이 온라인과 모바일 시장에서 경쟁할 수 있도록 하기 위해 만들어졌습니다.
이 제품은 2011년 이전에 완전한 버전이 출시될 예정입니다.

OTA 호텔은 데이터베이스 서버와 php 페이지를 관리하는 웹서버 이 단순한 2단계의 구성입니다.

코드 레이아웃

코드는 기능과 페이지로 나누어집니다.
주요 기능 파일은 다음과 같습니다:

  • configuration.inc.php - 구성 정의
  • hotel_defs.php - constants
  • login_check.inc.php - 단순한 로그인 확인 기능(본래의 hotelmis로부터)
  • lookupfunctions.php
  • queryfunctions.php - 기능을 만드는 데이터베이스 문의
  • functions.php - 작동에 대한 기능의 주요 목록
  • webtoPDF.php - html을 PDF로 변환 및 시작하는 기능

모든 다른 php 파일은 바로 html을 출력하고 호텔 페이지를 나타내야 합니다.

언어 – 번역

php 페이지 파일의 도입부에는 3줄의 코드가 있습니다. include_once("lang/lang_en.php");
$lang = get_language();
load_language($lang);

이 3줄의 코드는 최소한 영어로 된 버전이 각 웹 페이지에 로딩될 것을 확실히 해줍니다.
이들은 올바른 언어로 옮겨질 수 있게끔 모든 파일의 도입부에 위치해야 합니다.

언어 설정은 언어 표에 이용가능한 언어 목록과 함께 호텔 설정 페이지를 통한 데이터베이스에 설정되어 있습니다. 언어를 추가하기 위해서는, 언어 표가 수정되어야 합니다. 예를 들어,

lang - 국제 언어 코드
Description - 영어 설명
LocalDescription - UTF-8 암호화된 언어 버전
active - 0 이용은 가능하지만 선택은 불가, 1 활성화되어 있고 언어 파일이 존재

INSERT INTO `languages`(`lang`,`Description`,`LocalDescription`,`active`) VALUES ( 'my','Myanmar (Burmese)','Myanmar (Burmese)','0');

새로운 언어 파일을 생성하려면, lang/lang_en.php를 복사하여 lang/lang_.php로 하고, 파일을 편집하십시오.
데이터베이스를 업데이트하고 활동적인 상태를 1로 설정하십시오. 그리고 난 후 호텔 설정 페이지에서 언어가 설정될 것입니다.
