A Simple Video Tutorial on How to Setup a Holiday


Holiday is used to create a room rate which contains holidays.
Click the "Holiday Setup" menu item to view the list of holidays.

Buttons:
Delete: To remove a record of the existing holiday
Add: To setup a holiday

To add a new holiday, fill out the following information then click the Add button.

Field Description
Holiday DescriptionThe description of holiday
CalendarThe date of the holiday
