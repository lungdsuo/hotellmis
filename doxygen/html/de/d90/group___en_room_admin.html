 Hotel Room Type Setup
 

Detailed Description

Room setup for the OTA Hotel database

A Simple Video Tutorial on How to Setup a Room


Note:
For Guest Houses with dorm rooms and individual sold beds, setup each room as a room type and each bed as a room of that room type.

The administration screen for adding/editing room and the items contained.
Each room can specify up to 4 OTA described beds, and define the OTA amenities contained in the room.
These amenities will be used when running OTA queries.
