Room Types setup for the OTA Hotel database

A Simple Video Tutorial on How to Setup a Room Type


The OTA specification does not create standard room types, these are Hotel specific. Room types are a collection of the standard list of amenity and bed types which are defined by the OTA.
A room type is a collection of rooms, every room can have a room type, but rooms themselves may have different OTA amenities specified, it is up to the hotelier to correctly classify which room type a room belongs to.
