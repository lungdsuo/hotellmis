Installation and setup Requirements

To install the OTA Hotel software the following configurations are supported.
WAMP

  • Windows
  • Apache
  • Mysql
  • PHP

or
LAMP

  • Linux
  • Apache
  • Mysql
  • PHP


Install and setup the your LAMP/WAMP server as per the apache/php websites http://www.apache.org and http://www.php.net

Using the mysqladmin tool create a database called hotelmis
Create the database and sample room data using hotelmis.sql
Create a database user eg hotelmis/hotelmis


Now edit the configuration.inc.php
Set the defines for DB, USER and PASSWORD

On the host MySQL server, run the hotelmis.sql and create the database tables and initial data.

Default account is "admin" password "password"

Modify the configuration.inc.php file and set the definitions:

HOST is the IP or hostname where the mysql database is installed.
define("HOST", "localhost");

PORT is the socket number, default is 3306
define("PORT", 3306);

USER is the login to the mysql database
define("USER", "hotelmis");

PASS is the password to the user
define("PASS", "hotelmis");

DB is the database name
define("DB", "hotelmis");

DEBUG turns on some debug printing informatin
define("DEBUG", 0);

TAXPCT is the default percentage for tax calculations
define("TAXPCT", 7);

SVCPCT is the service percentage for service fees
define("SVCPCT", 10);
