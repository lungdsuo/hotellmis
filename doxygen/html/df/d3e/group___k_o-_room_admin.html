 Hotel Room Type Setup
 

Detailed Description

OTA 호텔 데이터베이스에 대한 객실 설정

객실 관리


객실 추가/편집과 물품이 포함된 관리 화면.
각 객실은 OTA에 설명된 최대 4가지의 침대 종류를 설정할 수 있고, 객실에 포함된 OTA 물품을 정할 수 있습니다.
이 물품은 OTA 문의를 실행할 때 사용될 것입니다.
